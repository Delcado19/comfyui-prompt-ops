# Prompt Builder

ComfyUI Prompt Ops includes an interactive prompt builder implemented with Espanso.

---

# Trigger

```
:prompt
```

Typing this trigger opens an interactive selection form.

---

# Builder Fields

The builder includes the following prompt components:

- Context
- Characters
- Scene
- Camera
- Lighting
- Style
- Quality
- Negative prompts
- NSFW modifiers

Each field provides predefined options.

---

# Output

The builder combines the selected components into a final prompt.

Example output:

```
portrait photo of a woman, cinematic lighting, close-up shot,
high detail skin texture, ultra quality
```

The generated prompt is inserted directly at the cursor position.

---

# Builder Source

The builder configuration is stored in:

```
snippets/zz_prompt_builder.yml
```

This file is automatically generated.

Generator script:

```
scripts/generate_prompt_builder.ps1
```

---

# Builder Generation

The prompt builder is regenerated automatically when running the development pipeline:

```powershell
.\scripts\dev.ps1
```

This ensures the builder always reflects the current snippet library.
